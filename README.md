#D Shell
##About
The D Shell.  
This is UNIX Shell Enviromnent "D Shell".  
Bash like syntax, but this is writteln in D language.  
  
  
##Installation
  % git clone https://github.com/alphaKAI/dsh
  % cd dsh
  % dub build
  % mkdir config
  
##Futures
    * JavaScript based ShellScript(D Shell contain Higgs JavaScriptVM)
